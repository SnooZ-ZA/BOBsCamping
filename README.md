# Install Instructions:
Extract files

Add rprogress folder to server resources

Import items.sql to database

Add to server.cfg

ensure rprogress

ensure BOBsHunt

# Use:
Hunt and skin Boar, Mountain Lion, Deer, Coyote and Rabbit for pelts to sell.

Knife needed to skin animal.
